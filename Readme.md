Aplicación web interactiva para prepararse para el examen de certificación **GitHub Copilot (GH-300)**.

## 📋 Sobre el proyecto

Esta herramienta de estudio fue **generada por Claude (Anthropic)** a partir de recursos de preparación recopilados de internet y proporcionados como material de referencia.

### Recursos utilizados

Los materiales base fueron investigados y compilados por **Max Carrión (NTT DATA)** e incluyen:

- **Guía de Consejos (Cheat Sheet)** — Mnemónicos, trampas frecuentes y respuestas flash

Estos recursos fueron encontrados en internet y utilizados como base para generar contenido adicional de práctica.
